## System Requirements for Odigos
To ensure optimal performance and compatibility when deploying Odigos, your environment must meet the following system requirements:

**Kubernetes Version**
Minimum Version: Odigos requires Kubernetes version 1.23.0 or higher.

<Info>
  Odigos Enterprise data collection for Go applications is powered by eBPF, which requires a Linux kernel version of 5.4.0 or higher.
</Info>

**Operating System**
Supported Platforms: Odigos operates on Kubernetes clusters running on nodes with a Linux kernel version of 5.4.0 or later.
