## System Requirements for Odigos
To ensure optimal performance and compatibility when deploying Odigos, your environment must meet the following system requirements:

**Kubernetes Version**
Minimum Version: Odigos requires Kubernetes version 1.20.3.0 or higher.

**Operating System**
Supported Platforms: Odigos operates on systems with an underlying Linux kernel version of 5.4.0 or higher.
